# Project Ascensus

**Математическое моделирование старта многоступенчатой ракеты**  
с учётом динамики расхода топлива, сброса пустых баков и оптимизации распределения масс топлива.

### О проекте

Проект реализует полную численную модель вертикального старта ракеты переменной массы.  
Модель учитывает:
- непрерывное выгорание топлива каждой ступени,
- автоматическое отделение пустых баков,
- изменение тяги двигателя с высотой,
- аэродинамическое сопротивление и переменную гравитацию,
- оптимизацию распределения топлива по ступеням для максимальной конечной скорости.

Позволяет сравнивать эффективность многоступенчатых схем и оценивать, при каких характеристиках двигателей становится возможен вывод полезной нагрузки одной или двумя ступенями.

### Основные возможности
- Объектно-ориентированная модель (`Rocket`, `Stage`, `Simulator`)
- Численное интегрирование методом Рунге-Кутты 4-го порядка (RK4)
- Оптимизатор распределения масс топлива (`RocketOptimizer`)
- Анализ реальных и гипотетических двигателей
- Автоматическое сохранение графиков и таблиц результатов

### Требования
- **Python 3.12**
- Библиотеки (устанавливаются одной командой):
  ```bash
  pip install -r requirements.txt
  ```

### Запуск
1. Откройте проект в среде разработки (VS Code / PyCharm и др.).
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите главный файл:
   ```bash
   python main.py
   ```

Результаты (графики и таблицы) сохраняются в папке `results/`.

### Цель проекта
Количественно ответить на классический вопрос ракетодинамики:  
**Насколько эффективнее многоступенчатые схемы и при каких современных характеристиках двигателей можно обойтись одной или двумя ступенями?**
